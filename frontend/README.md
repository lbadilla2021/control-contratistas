# ControlDoc Frontend

Next.js 14 (App Router) frontend scaffold for ControlDoc.

## Scripts
- `npm run dev` – ejecuta el entorno de desarrollo
- `npm run build` – genera el build de producción
- `npm start` – sirve el build generado
- `npm run lint` – ejecuta las reglas de linting

## Rutas iniciales
- `/` – página de login (pública)
- `/dashboard` – dashboard vacío protegido por middleware (requiere cookie `token`)
